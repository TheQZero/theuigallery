The Pagination

  • Allows for access to pages of data.
  • Usually for information that is repetitive, such as search results for Google.com
  • Useful for forums and databases.

The Modal Window

  • Can be programmed to pop up when needed.
  • Used for login pages, or advertisments.

The Spinner

  • Used to show when a page is loading, especially when it is waiting for more data to become available.
  • Provides the illusion of responsivness, leading to a better user experience.

The Card

  • A compact rectangle that usually displays an image and some text.
